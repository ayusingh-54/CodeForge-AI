About CodeForge AI

CodeForge AI blends large language models with tooling orchestration to help you write, review and ship code. This frontend demonstrates how you can embed an interactive chat experience into your project and how to wire it to a backend API.

Highlights

  • Responsive layout: works on phones, tablets and desktop screens
  • Demo-mode: runs locally without any API keys for quick testing
  • Extensible: example hooks show how to call a backend for real AI responses
  • Accessible: ARIA labels and keyboard shortcuts (Enter to send, Shift+Enter newline)

Quick setup

  1. Open Frontend/index.html in your browser to try the demo.
  2. To connect to a server, follow the snippet in Frontend/README.md.
